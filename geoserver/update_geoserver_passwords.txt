Вот Python-скрипт. Он:

- Читает конфигурацию из переменных окружения.
- Ищет все datastore.xml файлы в директории.
- Извлекает имя базы данных из XML.
- Поиск пароля в словаре PASSWORDS.
- Обновляет пароль в хранилище GeoServer через REST API.
- Выводит логи и статусы.

Как использовать:
Сохраните скрипт в файл, например: update_geoserver_passwords.py
Дайте права на выполнение:
bash
```sh
chmod +x update_geoserver_passwords.py
```
Установите зависимости:
bash
```sh
pip install requests
```
Запустите с переменными окружения:
bash
GEOSERVER_URL=http://your-geoserver:8080/geoserver \
GEOSERVER_WORKSPACES_DIR=/data/geoserver/data_dir/workspaces \
GEOSERVER_USER=admin \
GEOSERVER_PASS=geoserver \
python3 update_geoserver_passwords.py

✅ Особенности:
- Использует requests для REST API.
- Парсит XML с помощью xml.etree.ElementTree (без внешних библиотек).
- Поддерживает обновление пароля в connectionParameters.
- Поддерживает логирование и обработку ошибок.
- Использует glob для поиска datastore.xml по всем директориям.
